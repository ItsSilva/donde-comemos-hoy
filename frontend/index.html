01

Del gusto individual al perfil N del grupo.

Cada integrante califica cinco dimensiones: picante, dulce, salado, vegetariano y carne. El sistema transforma esos datos en un vector grupal.

02

El algoritmo compara el grupo con restaurantes.

El frontend envía el grupo a la API Flask. El backend aplica similitud coseno y devuelve los restaurantes ordenados por compatibilidad.

03

La recomendación muestra justicia, no solo ranking.

Además del score grupal, se visualiza la satisfacción mínima para entender quién queda menos representado en la decisión.
